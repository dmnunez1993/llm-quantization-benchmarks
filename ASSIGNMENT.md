Design and execute an experiment of your choice, following these steps:

- Choose a question to investigate using a scientific experiment.
- Design the experiment (choose dependent and independent variables, data collection protocol, and data analysis protocol) to investigate the chosen question.
- Execute the experiment following the data collection protocol.
- Analyze the data obtained, following the data analysis protocol.
- Take note of any surprising or unsurprising findings, and prepare a conclusion for the experiment.

The student is expected to analyze the data collected through the establishment of statistical hypothesis prior to the experiment, and the calculation of statistical inference metrics after the experiment.

In addition, the students should analyze whether the data follows the assumptions of the model, and calculate the sample size necessary for the desired power.

If it is not possible to obtain the desired sample size, the student must calculate the power of the experiment with the available sample size, and comment on how the difference of theoretical sample size and actual sample size influences the conclusions about the experiment.

You must submit TWO files:
- A PDF file with your report in text format.
- A ZIP file with the data and scripts necessary to reproduce your data analysis. (Please note that this zip file should contain the data and scripts used to reproduce the *statistical data analysis*: experimental result, statistical estimators and tests, tables, figures, etc)
